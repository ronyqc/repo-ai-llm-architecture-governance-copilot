# Descripción

Define versionamiento de APIs.

# Problema

Cambios rompen clientes.

# Recomendación

Usar versionado en URL.

# Cuándo aplica

APIs públicas.

# Cuándo no aplica

APIs internas controladas.

# Impacto arquitectónico

Mejora compatibilidad.

# Ejemplo

/api/v1/customers
