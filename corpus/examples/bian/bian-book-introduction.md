# Descripción general

BIAN es un framework de arquitectura de referencia para la industria bancaria que define funciones de negocio, servicios e interacciones de forma estandarizada.

# Propósito de negocio

Reducir costos de integración, mejorar interoperabilidad y habilitar agilidad en bancos.

# Capacidades principales

- Definición de servicios bancarios estándar
- Modelo semántico común
- Integración entre sistemas y proveedores

# Relación con otros dominios

Actúa como “lenguaje común” entre estándares financieros (ISO, regulaciones, etc).

# Aplicabilidad en arquitectura

- Diseño de arquitectura empresarial bancaria
- Definición de capacidades
- Modelado de APIs
